## Welcome to Hexo

This is your first post. Edit or delete it, then start writing.

### Quick Start

**Create a new post:**

```bash
hexo new "My New Post"
```

**Run the server:**

```bash
hexo server
```

**Generate static files:**

```bash
hexo generate
```

**Deploy to remote:**

```bash
hexo deploy
```

### Folder Structure

```
.
├── _config.yml       # Site configuration
├── source/           # Content folder
│   └── _posts/       # Blog posts
├── themes/           # Theme folder
└── scaffolds/        # Post templates
```

> For more details, check the [Hexo documentation](https://hexo.io/docs/).
