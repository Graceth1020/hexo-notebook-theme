Welcome

Welcome to the Notebook theme for Hexo.

This theme focuses on clean typography, a tag-based tree navigation system, and a minimal reading experience.

Features

  • Tag Tree — Browse posts through a hierarchical tag structure on the /tags page
  • Responsive Design — Works on desktop, tablet, and mobile
  • Clean Typography — Focus on what matters: your content
  • Fast — Lightweight with no unnecessary dependencies

Getting Started

  1. Install the theme
  2. Configure _config.yml
  3. Start writing posts

Happy blogging!
